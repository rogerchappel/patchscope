# Roadmap

## Now

- Local unified diff parsing.
- Deterministic Markdown and JSON reports.
- Secret-like addition detection.
- Review risk classification.
- Test suggestion heuristics.

## Next

- Config file support for repository-specific risk thresholds.
- More ecosystem-aware test suggestions.
- SARIF export for tools that can ingest local static-analysis evidence.
- Optional baseline comparison for recurring generated-file churn.

## Later

- Language-specific heuristics that remain deterministic and offline.
- Editor/task-runner integrations.
- Richer docs around agentic coding workflows.

## Non-goals

- Hosted service.
- Telemetry.
- LLM review.
- Automatic code modification.
